# 🚀 RR_CareerAI - India's Smart Career Roadmap & Risk Analyzer

**CareerAI** is a modern, AI-powered career guidance platform specifically tailored for the Indian job market. It helps students and professionals navigate the rapidly changing landscape of work by providing data-driven roadmaps, AI risk assessments, and career comparisons.

## ✨ Key Features

- 🤖 **AI Risk Analyzer:** Understand which parts of a career are likely to be automated and how to stay relevant.
- 🛣️ **Personalized Roadmaps:** Step-by-step phases with timelines and skill prioritization.
- ⚖️ **Career Comparison:** Side-by-side analysis of roles including salary (INR/LPA), growth, and demand.
- 💡 **Smart Suggestions:** Get career matches based on your background, interests, and lifestyle values.
- 🇮🇳 **Local Context:** All insights and salary data are focused on the Indian economy and job market.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS (Modern, Responsive UI)
- **AI Intelligence:** Llama 3.3 (via Groq Cloud API)
- **Deployment:** Vercel

---

### Prerequisites

Because this project uses a private API key for security and billing reasons, the **cloned repository will not work out of the box.** To run it locally, you will need:

- [Node.js](https://nodejs.org/) installed.
- A **Groq API Key** (Get one for free at [console.groq.com](https://console.groq.com/)).
